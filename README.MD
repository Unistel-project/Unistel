# UNISTEL - Hostel Booking System

## Project Overview

UNISTEL is a web-based Hostel Booking System designed to help students search, compare, and book hostel accommodations online before arriving at campus. The system aims to reduce the challenges students face when looking for suitable hostels, such as long travel distances, traffic congestion, limited information, and delays in securing accommodation.

## Problem Statement

Students often struggle to find and book suitable hostel accommodation due to the lack of a centralized and efficient booking platform. This results in inconvenience, wasted time, and difficulty accessing accurate hostel information.

## Project Objectives

### Main Objective

To develop an online hostel booking system that enables students to easily search for and reserve hostel accommodation.

### Specific Objectives

* Provide a centralized platform for hostel listings.
* Allow students to search and compare available hostels.
* Enable online hostel booking and reservation.
* Improve access to accurate hostel information.
* Reduce the time and effort required to secure accommodation.

5. Run the application on a local server.
